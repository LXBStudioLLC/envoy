# Code of Conduct

The Envoy project uses the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. The full text is at the link above and is reproduced in many open-source projects.

## Summary

We want participation in the Envoy community to be a respectful, welcoming, harassment-free experience for everyone, whatever their background, identity, or experience level. Maintainers will enforce this code fairly and consistently.

## Scope

This Code applies in all project spaces: GitHub issues and pull requests, the repository's Discussions tab, Discord or Slack if those ever exist, and any other forum officially tied to the project. It also applies when someone is representing the project in public.

## Reporting

If you see or experience behavior that breaks this Code, please report it privately:

- Email `LXBStudioLLC@gmail.com` with the subject line `[envoy conduct]`.

We'll review and respond within 7 days. We keep reports confidential and won't share your identity with the subject of the report without your consent.

## Enforcement

Maintainers follow the [Contributor Covenant Enforcement Ladder](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines), which runs from a correction to a warning to a temporary ban to a permanent ban, depending on the nature and severity of the violation.
