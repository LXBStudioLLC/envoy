# Code of Conduct

The Envoy project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. The full text is available at the link above and is reproduced in many open-source projects.

## Summary

We pledge to make participation in the Envoy community a respectful, welcoming, and harassment-free experience for everyone, regardless of background, identity, or experience level. Maintainers will enforce this code fairly and consistently.

## Scope

This Code applies in all project spaces — GitHub issues and pull requests, the repository's Discussions tab, Discord/Slack if and when those exist, and any other forum officially associated with the project. It also applies when an individual is representing the project in public spaces.

## Reporting

If you observe or experience behavior that violates this Code, please report it privately:

- Email **`LXBStudioLLC@gmail.com`** with the subject line `[envoy conduct]`.

Reports will be reviewed and responded to within 7 days. We commit to confidentiality for the reporter; we will not share your identity with the subject of the report without your consent.

## Enforcement

Maintainers will follow the [Contributor Covenant Enforcement Ladder](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines) — correction, warning, temporary ban, or permanent ban — depending on the nature and severity of the violation.
